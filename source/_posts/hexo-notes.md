### 更换电脑之后重新部署

```shell
git clone ${your-repository}
cd ${your-repository}
npm install hexo
npm install
npm install hexo-deployer-git
```

### 常用命令

```shell
# Starts a local websites.
hexo s
# Creates a new post.
hexo n "post-name"
# Cleans cache files.
hexo clean
# Generates hexo public files.
hexo g
# Deploys hexo websites.
hexo d
```
